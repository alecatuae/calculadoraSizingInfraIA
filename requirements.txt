# Calculadora de Sizing de Inferência LLM
# 
# Este projeto usa apenas a biblioteca padrão do Python (stdlib).
# Não há dependências externas necessárias.
#
# Requisitos mínimos:
# - Python 3.8+
#
# Módulos stdlib utilizados:
# - argparse (CLI arguments)
# - json (parsing de dados)
# - math (cálculos matemáticos)
# - sys (system interaction)
# - dataclasses (opcional, estruturas de dados)
# - subprocess (test_sizing.py)
#
# Para executar:
#   python3 sizing.py --help
#
# Para testar:
#   python3 test_sizing.py
